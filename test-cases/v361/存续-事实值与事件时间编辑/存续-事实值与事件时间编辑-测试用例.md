## 3. 通用规则（全产品适用）

1. **可编辑时机：** 仅 **触发事件结束后** 出现 ✏️；事件时间可改（见 UI 原型）。
2. **每条触发事件至少覆盖：** `*-FACT-*`（各事实值字段）、`*-TIME-*`（事件时间）、`*-NEG-*`（未结束不可编辑）。
3. **输入校验 / 权限 / 弱网：** 见 [00-通用](./cases/00-通用-UI与负向.md)。

## 5. 通过准则

- P0 **100%** 通过（含各产品各触发事件 `*-FACT-001`、`*-TIME-001`、`*-NEG-001`）。
- **事件未结束可编辑** → Blocker。
- **可编辑字段超出矩阵** → Major。
- **SCN 两行 FF Cash 字段混淆** → Major。
