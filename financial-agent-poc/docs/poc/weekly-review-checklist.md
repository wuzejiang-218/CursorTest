# 周评审清单（PoC 12 周内复用）

**会议时长建议**: 45–60 分钟  
**固定参与**: 测试 Owner、后端/平台、业务/风控代表（可轮流）

## 1. 进度对齐

- [ ] 本周计划完成项 vs 实际
- [ ] 阻塞项（环境、账号、契约、数据）与负责人
- [ ] 范围变更是否触发验收指标调整

## 2. 质量与指标

- [ ] 自动生成+执行用例占比（对照 `01-scope-and-acceptance.md`）
- [ ] 回归耗时对比基线（`03-manual-baseline-template.md`）
- [ ] 误报/漏报样例各 ≥3 条复盘

## 3. 平台与工程

- [ ] 流水线稳定性（失败是否为环境噪声）
- [ ] 审计日志是否完整（run_id、步骤、版本）
- [ ] 模型成本与并发是否在 `config/governance.yaml` 阈值内

## 4. 合规与安全

- [ ] 脱敏是否覆盖新增日志/报告字段
- [ ] 权限与密钥轮换是否有到期项
