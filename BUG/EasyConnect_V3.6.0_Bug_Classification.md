# EasyConnect V3.6.0 Bug 归因分类（按人 × 类型）

> 分类口径
> - **不应该出现的 Bug**：低级缺陷类，包括字段映射 / 字段缺失 / 单位 / 校验缺失 / 排序规则 / 空指针 / 邮件模板缺字段 / 基础产品类型限制等。开发自测或 CR 阶段本应拦截。
> - **可能没想到的 Bug**：复杂业务联动 / 多状态组合 / 时序边界 / 并发 / 跨系统交互 / 历史权限链路等，设计 / 需求 / 测试阶段不易完整覆盖。
> - **其他**：不予解决 / 外部原因 / 重复 Bug，未计入分类。

## 风险提示

1. **胡明明 11 / 15 为"不应该出现"**，集中在上架失败（4 条）、字段映射（3 条）、列表排序（3 条），属于开发自测可覆盖范围，建议在提测前增加产品级冒烟用例。
2. **蒋杰 9 / 16 为"不应该出现"**，邮件模板类占 4 条（8069 / 8089 / 8091 / 8093），建议沉淀一份邮件字段 / 单位 / 模板布局 checklist，发版前强制校验。
3. **肖赣丰"没想到"占 8 条**，主要是订单 ↔ Tapon 联动、批量审批并发场景，建议针对"主单修改 / 批量同意 / 跨订单同步"三条主线补端到端用例。
4. **彭超 6 / 11 落入"其他"**（不予解决 / 外部 / 重复），需关注重复 Bug 链（8023→8020、8052→8036）是否同一根因导致重复提单。
5. 整体 **32 / 61 ≈ 52% 为"不应该出现"** 的低级缺陷，建议下个迭代将"字段映射 + 邮件模板 + 列表排序"三类纳入 CR 必查项。
