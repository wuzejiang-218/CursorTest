## 8.1 同步回复申请单字段

| 字段 | 说明 |
|------|------|
| request_id | 申请单 ID |
| inquiry_id | 询价 ID |
| issuer_id | 被申请 Issuer |
| reply_id | 被申请价格回复 ID |
| org_id | 机构 ID |
| buyside_id | 机构 Buyside（与 Issuer 表 Matched Last Look Buyside 对应） |
| applicant_user_id | 发起机构用户（手动时） |
| status | 待确认 / 已确认 / 已拒绝 / 已失效 |
| trigger_mode | `manual` / `automatic` |
| target_best_price | 创建时最优价快照 |
| confirmed_price | 实际确认同步价 |
| confirm_channel | 邮件 / 自动 / 手工（Issuer Reply Mode） |
| last_look_setting_snapshot_id | 机构配置快照 |
| issuer_last_look_row_id | Issuer 侧 Buyside 配置行 ID |
| validity_minutes | 来自机构 Last Look Validity Period |
| created_at / confirmed_at / expired_at | 时间戳 |

## 8.2 机构配置存储字段

| 字段 | 对应界面 |
|------|----------|
| matched_last_look_issuer | Matched Last Look Issuer |
| triggering_mode | Manual / Automatic |
| auto_product | Product |
| auto_trigger_form | Trigger form |
| best_price_difference_value | Set Best Price Difference Value |
| validity_period_minutes | Last Look Validity Period |
| authorized_user_ids | User authorized to initiate last look |

## 8.3 Issuer 配置存储字段

| 字段 | 对应界面 |
|------|----------|
| issuer_id | Issuer 主档（如 CACIB PG） |
| matched_last_look_buyside | Matched Last Look Buyside |
| match_price_time_minutes | Match Price Time |
| reply_last_look_mode | Reply Last Look Mode（View 子配置：email / automatic / manual） |
| price_validity_period_minutes | Price Validity Period |
| enabled | 行是否启用 |

## 8.4 待产品确认项

- [ ] 自动模式下机构授权用户是否可「补发」手工申请
- [ ] Trigger form 除「By Respond Time in Each Order X Minutes」外是否还有其它枚举
- [ ] 机构 Matched Issuer 与 Issuer Matched Buyside 多选时的匹配规则
- [ ] Best Price Difference Value 未配置时是否等同于不限制
- [ ] Match Price Time 与机构 Trigger form / Validity Period 的优先级与起算点
- [ ] Issuer Price Validity Period 为 `--` 时的默认行为
- [ ] Reply Last Look Mode 子页面完整字段清单（View 弹窗）
