# 07 验收标准

以下采用 **Given / When / Then**，供测试用例拆分。

---

## 7.1 机构配置

**AC-C01 自动模式按条件触发**

- **Given** 机构配置：Mode=Automatic，Product=FCN，Trigger=每笔订单回复满 5 分钟，Matched Issuer=CACIB，Validity=20 分钟
- **When** FCN 询价中 CACIB 回复满 5 分钟且报价非最优、价差未超 Difference Value
- **Then** 系统自动创建待确认申请；20 分钟内未确认则已失效

**AC-C02 价差过大不自动触发**

- **Given** 已配置 Best Price Difference Value=10bp，CACIB 与最优价差距 15bp
- **When** 其余自动条件满足
- **Then** 不自动创建申请

**AC-C03 手动模式仅授权用户可发起**

- **Given** Mode=Manual，授权用户列表含 User U1
- **When** U1 对非最优 CACIB 发起申请
- **Then** 创建成功；未授权用户 U2 发起则被拒绝

**AC-C04 手动模式未配置授权用户**

- **Given** Mode=Manual，授权用户为空
- **When** 任意用户尝试发起
- **Then** 拒绝并提示配置 Last Look Setting

## 7.2 Issuer 配置

**AC-I01 双向匹配成功**

- **Given** 机构 CAI HK 配置 Matched Issuer=CACIB；Issuer CACIB PG 配置 Buyside 行 CAI HK，Reply Mode=邮件
- **When** 满足机构自动/手动发起条件
- **Then** 创建申请并向 CACIB 发送邮件确认；记录 issuer_buyside_row_id

**AC-I02 Buyside 未在 Issuer 表配置**

- **Given** 机构 CAI HK 配置 Matched Issuer=CACIB；Issuer 侧无 CAI HK 行
- **When** 机构尝试触发 Last Look
- **Then** 不创建申请；提示 Issuer/机构配置不匹配

**AC-I03 Match Price Time 未配置**

- **Given** Issuer 对 CAI SG 行 Match Price Time = `--`
- **When** 机构 CAI SG 发起询价且满足其它条件
- **Then** 按产品默认时间规则处理或不允许发起（以实现为准，需与产品对齐）

**AC-I04 Reply Mode 自动确认**

- **Given** Issuer 对 CAI HK 的 Reply Last Look Mode = Automatic（View 内配置）
- **When** 系统创建待确认申请
- **Then** 按规则自动确认改价；记录 confirm_channel=automatic

**AC-I05 Price Validity Period 过期**

- **Given** Issuer 配置 Price Validity Period=15 分钟，回复已超过 15 分钟
- **When** 机构侧创建或 Issuer 尝试确认
- **Then** 申请已失效或不可确认；价格与最优价不变

## 7.3 发起与确认

**AC-01** 非最优 Issuer 可发起（手动或自动）→ 待确认，记录 `trigger_mode=manual|automatic`

**AC-02** 最优 Issuer 不可发起

**AC-03** 重复发起拦截

**AC-04～AC-07** Issuer 邮件/自动/手工确认、拒绝

**AC-08** 超过机构 **Last Look Validity Period** → 已失效

**AC-09～AC-12** 最优价变化、询价关闭、新回复、并发确认

**AC-13** Lastlook 展示比价、申请状态、机构触发模式、Issuer Reply Mode 摘要
